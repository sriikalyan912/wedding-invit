PLACE YOUR IMAGES IN THIS FOLDER
================================

The site reads image paths from  js/config.js.  Add your photos here
using these exact filenames (or change the names in config.js):

  hero.jpg        -> full-screen background behind the couple's names
                     (landscape, ~1920x1280 looks best)

  gallery-1.jpg   -> photos shown in the "Moments" gallery grid
  gallery-2.jpg      (square crops look best, ~800x800)
  gallery-3.jpg
  gallery-4.jpg
  gallery-5.jpg
  gallery-6.jpg

Add or remove as many gallery images as you like — just keep the list
in config.js (the `gallery: [...]` array) matching the files here.

Until you add real photos, the site shows tasteful placeholder cards,
so nothing looks broken.

TIP: compress images (e.g. tinypng.com) before uploading to S3 so the
page loads fast on mobile.
